# Fluxo do Usuário

1. Usuário faz login
2. Cadastra dados da empresa
3. Inicia nova emissão
4. Informa:
   - UF origem
   - UF destino
   - Operação
   - Finalidade
   - Regime tributário
5. Sistema consulta regras fiscais
6. Sistema retorna:
   - CFOP
   - CST/CSOSN
   - ICMS
   - IPI
   - Observações
7. Usuário salva a orientação
